## 🔍 Overview

**VYTALIX AI** is an AI-powered health prediction system that analyzes patient vital signs in real-time and predicts potential medical conditions using an ensemble Machine Learning model. The system provides instant clinical insights, actionable precautions, ECG waveform simulations, and downloadable PDF reports — all through a sleek, modern web dashboard.

| Feature | Description |
|---------|-------------|
| 🧠 **ML-Powered Prediction** | Ensemble model (Random Forest + Gradient Boosting + MLP) trained on 100K synthetic patient records with **99.78% accuracy** |
| 📊 **Interactive Dashboard** | Dark-themed, responsive UI with live vital sign cards, trend charts, and radar comparisons |

## 🛠️ Tech Stack

### Backend
- **Python 3.x**
- **Flask** — REST API server
- **Flask-CORS** — Cross-origin support
- **scikit-learn** — ML model training (RandomForest, GradientBoosting, MLP, VotingClassifier)
- **NumPy & Pandas** — Data processing

### Frontend
- **HTML5 / CSS3 / JavaScript** — Single-page dashboard
- **Chart.js** — Interactive charts (trend lines, radar, doughnut)
- **jsPDF** — Client-side PDF report generation
- **Google Fonts** — Space Grotesk & JetBrains Mono typography

### ML Model
- **Algorithm**: Soft Voting Ensemble (Random Forest + Gradient Boosting + MLP Neural Network)
- **Training Data**: 100,000 synthetic patient records with 15 vital features
- **Accuracy**: ~99.78%
- **Features**: 15 vital parameters including heart rate, SpO₂, blood pressure, glucose, ECG intervals, etc.

## 🚀 Installation & Setup

### Prerequisites
- Python 3.8 or higher
- pip (Python package manager)

### Step 1: Install Dependencies

```bash
pip install flask flask-cors numpy pandas scikit-learn
```

### Step 2: Train the ML Model

```bash
python train_model.py
```

This will:
- Generate 100,000 synthetic patient records
- Train the ensemble ML model (Random Forest + Gradient Boosting + MLP)
- Save `vital_model.pkl` and `model_meta.json`

### Step 3: Run the Application

```bash
python flask_api_backend.py
```

The server will start and automatically open the dashboard in your browser at:

> **🌐 http://127.0.0.1:5050**

## 📖 Usage

### Running an Analysis
1. Enter the **Patient Name** in the Patient Info section
2. Fill in **Demographics** (Age, Gender, BMI, Respiratory Rate)
3. Enter **Core Vitals** (Heart Rate, SpO₂, Temperature, Blood Pressure, Glucose)
4. Enter **ECG Parameters** (PR Interval, QRS Duration, QT Interval, ST Elevation)
5. Enter **Blood Panel** values (Cholesterol)
6. Click **"⚡ Run AI Health Analysis"**

### Quick Sample Data
Use the preset buttons to instantly load sample data:
- **Normal** — Healthy patient readings
- **HTN** — Hypertension case
- **ECG** — Arrhythmia case
- **Critical** — Multi-organ risk case

### Downloading PDF Report
After running an analysis, click **"📄 Download Patient Report as PDF"** to generate a professional report containing:
- Patient demographics & name
- AI diagnosis with confidence percentage
- Complete vital signs table with status indicators
- Abnormal vital alerts with precautions
- Top ML predictions
- Clinical recommendations

### Dashboard Tabs

| 📊 Dashboard | Main input form, vital cards, diagnosis results, trends, and radar chart |
| 🔬 Analysis | AI clinical analysis, medical report comparison, vitals comparison table |

## 🤖 ML Model Details

### Training Pipeline (`train_model.py`)
1. Generates 100K synthetic patient records with realistic vital sign distributions
2. Applies pathological rules to label conditions (Hypertension, Bradycardia, etc.)
3. Trains three models: Random Forest (150 trees), Gradient Boosting (100 estimators), MLP Neural Network (256-128-64)
4. Combines into a Soft Voting Ensemble for probability-based prediction
5. Achieves ~99.78% test accuracy

### Feature Importance (Top 5)
| Feature | Importance |
|---------|-----------|
| Diastolic BP | 36.96% |
| Systolic BP | 26.74% |
| Heart Rate | 9.86% |
| Glucose | 9.71% |
| PR Interval | 7.24% |

### API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | Serves the dashboard UI |
| POST | `/api/predict` | Runs ML prediction on vitals |
| POST | `/api/trends` | Generates 24h trend data |
| POST | `/api/ecg` | Simulates ECG waveform |
| GET | `/api/meta` | Returns model metadata |
| GET | `/api/health` | Health check endpoint |
